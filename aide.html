🚀 Demarrage rapide
Gestionnaire de projets local inspire de Monday.com, specialise pour les projets Arduino, Raspberry Pi, impression 3D et logiciels.
1
Lancer le serveur
Double-cliquer sur start.bat. Une fenetre noire s'ouvre — ne pas la fermer tant que tu utilises l'appli.
2
Ouvrir dans le navigateur
Le navigateur s'ouvre automatiquement sur http://localhost:8742
3
Creer un projet
Cliquer sur + Nouveau projet ou appuyer sur N. Un dossier est cree automatiquement dans data/Nom du projet/
4
Ajouter des taches
Cliquer sur + Tache dans l'en-tete du projet ou appuyer sur T
Structure des donnees : Chaque projet a son propre dossier dans data/Nom du projet/ avec projet.json, taches.json et les sous-dossiers images/, stl/, cura/, docs/.
👁 Les 5 vues
Chaque vue offre une perspective differente sur tes projets et taches.
📋 Tableau B
Vue principale. Affiche tous les projets avec leurs taches. Colonnes : Statut, Priorite, Deadline, Temps (chrono), Estime, Passe. En-tete de projet collant.
📁 Kanban K
4 colonnes : A faire, En cours, Termine, Bloque. Glisser-deposer les projets entre les colonnes.
📅 Calendrier
Vue mensuelle avec les deadlines. Navigation mois par mois. Cliquer sur un evenement pour ouvrir.
📈 Timeline / Gantt
Barres horizontales representant la duree. Ligne rouge = aujourd'hui. Necessite une date de debut et une deadline.
📊 Dashboard D
Statistiques globales + 5 graphiques avances en bas : temps passe, statuts, types, retards, avancement SVG.
📁 Gestion des projets
Chaque projet est stocke dans son propre dossier et peut avoir des taches, fichiers, composants, budget, schema et tags.
📝 Informations de base
  • 👁
    Nom — Aussi utilise comme nom de dossier dans data/
  • 📦
    Type — Arduino/RPI, Logiciel, Mixte, Divers
  • 🔴
    Statut — A faire, En cours, Termine, Bloque (clic sur la pill)
  • 🎯
    Priorite — Haute, Moyenne, Basse
  • 📅
    Dates — Date de debut et deadline
  • 📄
    Description — Notes, liens, objectifs
⚙ Boutons en-tete projet
  • 📁
    📁 Fichiers — Gestionnaire images/STL/Cura/docs
  • 🖨
    🖨 Imprimer — Impression de ce projet uniquement
  • 📄
    PDF — Export PDF du projet
  • 📊
    XLS — Export Excel du projet
  • 🔄
    Dupliquer — Copie le projet avec toutes ses taches
  • 🗃
    Archiver — Masque sans supprimer
  • ✏
    Modifier — Ouvre la modale edition
  • 🔥
    Supprimer — Suppression definitive
📈 Schema de cablage
Dans la modale projet, champ "Schema de cablage". Bouton Parcourir pour selectionner une image. Cliquer sur "Schema" dans l'en-tete pour agrandir en lightbox.
💰 Budget
Champs Budget estime et Budget reel dans la modale. Widget dans le Dashboard avec barre de progression. Le total des composants est additionne automatiquement.
🏷 Tags et liens
  • 🏷
    Tags — impression3D, electronique, mecanique, firmware, urgent...
  • 🔗
    Depend de (orange) — tri topologique automatique
  • 📄
    Reference (bleu) — lien informel entre projets
✅ Gestion des taches
Les taches ont deux niveaux de sous-taches. La colonne PASSE affiche le temps reel passe (chronometre).
📝 Champs d'une tache
  • 🔴
    Statut — clic sur la pill dans le tableau
  • 🎯
    Priorite — Haute, Moyenne, Basse
  • 📅
    Debut / Deadline
  • ⌛
    Estimation — format Xh YYmin
  • 🕐
    Chronometre — ▶ pour demarrer, colonne TEMPS
  • 🔔
    Rappel — Notification Windows date/heure
  • 📄
    Notes — Details techniques
  • 🏷
    Tags — dans la modale detail
  • 📷
    Pieces jointes — images base64, lightbox
🕐 Chronometre et temps
  • ▶
    Colonne TEMPS : bouton ▶ pour demarrer
  • ⏹
    Bouton ⏹ pour arreter et sauvegarder
  • 💾
    Sauvegarde auto toutes les 60 secondes
  • 📈
    Colonne PASSE : total reel (chrono + sous-taches)
  • 📊
    Total visible dans la barre du bas et le Dashboard
📄 Sous-taches (2 niveaux)
  • ➕
    Survol tache → bouton + Etape vert
  • ✅
    Cocher la parente coche toutes les sous-taches
  • 🔴
    Toutes les sous-taches terminees = parente terminee auto
  • ⚠
    Impossible de cocher la parente si sous-taches non finies
💬 Commentaires et pieces jointes
Dans la modale detail (bouton "Ouvrir" au survol) : commentaires horodates avec edit/suppression. Ctrl+Entree pour envoyer. Pieces jointes images avec lightbox et multi-selection.
Rappels : Autoriser les notifications via 🔔 dans la topbar. Avec le Service Worker, les rappels fonctionnent meme si la page est fermee (navigateur ouvert en arriere-plan).
📁 Gestionnaire de fichiers
Chaque projet a son propre dossier avec 4 sous-dossiers pour stocker les fichiers physiquement sur le disque.
📁 Structure des dossiers
  • 📷
    images/ — Photos, captures, visuels du projet
  • 🚬
    stl/ — Fichiers STL pour impression 3D
  • ⚙
    cura/ — Profils et projets Ultimaker Cura
  • 📄
    docs/ — Autres documents (PDF, textes...)
⇧ Ouvrir le gestionnaire
Dans l'en-tete du projet, cliquer sur le bouton 📁 bleu. Une modale s'ouvre avec 4 onglets correspondant aux sous-dossiers.
⇧ Upload de fichiers
  • 👉
    Selectionner l'onglet du type de fichier (Images, STL, Cura, Docs)
  • ⇧
    Cliquer sur la zone pointillee ou glisser-deposer
  • 📸
    Multi-selection possible
  • 📷
    Les images s'affichent en miniature cliquable (lightbox)
  • ↓
    Bouton de telechargement pour chaque fichier
  • 🗑
    Bouton de suppression pour chaque fichier
📁 Acces direct au dossier
Les fichiers sont stockes dans data/Nom du projet/images/ etc. Tu peux y acceder directement depuis l'Explorateur Windows pour copier/coller des fichiers manuellement.
🔧 Liste de composants
Chaque projet peut avoir une liste de composants avec quantite, fournisseur, prix et lien.
🔧 Acceder aux composants
Dans l'en-tete du projet, cliquer sur le bouton 🔧. Un compteur s'affiche quand des composants sont definis.
📋 Champs par composant
  • 📝
    Nom — Ex: Arduino Mega 2560
  • 🔢
    Quantite
  • 🏫
    Fournisseur — Ex: AliExpress, Amazon
  • 💰
    Prix unitaire — En EUR
  • 🔗
    Lien URL — Lien direct vers le produit
💰 Total automatique
Total calcule automatiquement (quantite x prix) visible en bas de la liste et dans le widget Budget du Dashboard.
⚓ Raccourcis clavier
Ces raccourcis fonctionnent quand le curseur n'est pas dans un champ de saisie.
Touche Action Details
NNouveau projetOuvre la modale de creation
TNouvelle tacheDans le projet courant (dernier survole)
DVue DashboardTableau de bord global
BVue TableauVue Board principale
KVue KanbanVue colonnes glissables
?Aide raccourcisListe des raccourcis dans l'appli
EchapFermerFerme toute modale ouverte
Astuce : Le bouton ? dans la topbar affiche aussi les raccourcis directement dans l'appli.
