Démarrage rapide

GP est un gestionnaire de projets local sans compte, sans cloud, sans abonnement. Conçu pour les Makers : Arduino, Raspberry Pi, impression 3D, électronique. Tes données restent sur ton PC et se sauvegardent automatiquement sur ton NAS.

1
Lancer le serveur
Double-cliquer sur start.bat. La fenêtre noire = serveur actif. Ne pas la fermer.
2
Ouvrir l'application
Le navigateur s'ouvre automatiquement sur http://localhost:8742
3
Créer un projet
Clic sur + Nouveau projet (vert, topbar) ou touche N
4
Ajouter des tâches
Clic sur + Tâche dans l'en-tête du projet ou touche T
5
Tutoriel interactif
Clic sur 🎯 Tutoriel dans la topbar — 9 étapes guidées avec spotlight
📁

Structure données : chaque projet a son dossier dans data/Nom du projet/ contenant projet.json, taches.json, images/, stl/, cura/, docs/. Archives ZIP dans archives/. Sauvegarde NAS automatique dans \\NAS\GestionProjet\.

Les 5 vues

Chaque vue offre une perspective différente sur tes projets. Navigation via la sidebar ou les onglets.

📋 Tableau B
Vue principale. Projets + tâches en tableau. Colonnes : Tâche, Statut, Priorité, Deadline, Temps H/Min, Estimé, Passé. En-tête de projet collant avec bouton ▶ collapse.
🗂 Kanban K
4 colonnes : À faire, En cours, Terminé, Bloqué. Drag & drop projets entre colonnes.
📅 Calendrier
Vue mensuelle avec deadlines. Navigation mois par mois. Clic sur un événement pour ouvrir le projet.
📈 Timeline / Gantt
Barres horizontales par durée estimée. Ligne rouge = aujourd'hui. Nécessite date de début et deadline.
📊 Dashboard D
Statistiques globales + 5 graphiques : temps passé, statuts, types, retards, avancement par projet.

Gestion des projets

Chaque projet est dans son propre dossier. Il peut contenir des tâches, fichiers, composants, budget, schéma de câblage et tags.

✏ Champs d'un projet
  • 📛
    Nom — utilisé comme nom de dossier dans data/
  • 📦
    Type — Arduino/RPI, Logiciel, Impression 3D, Mixte, Divers
  • 🔴
    Statut — À faire, En cours, Terminé, Bloqué, En attente
  • 🎯
    Priorité — Haute, Moyenne, Basse
  • 📅
    Dates — Début et deadline
  • 💶
    Budget — Estimé et réel en EUR
  • 📄
    Description — notes, objectifs, liens
  • 🏷
    Tags — impression3D, electronique, mecanique...
⠿ Drag & drop projets Nouveau
  • ⠿
    Poignée ⠿ visible à gauche de chaque en-tête projet
  • 🖱
    Glisser pour réordonner les projets dans la vue Tableau
  • 📋
    Le tri passe automatiquement en "Tri : Manuel"
  • 💾
    Ordre sauvegardé sur le disque immédiatement
  • 🔄
    Pour revenir au tri par date : sélectionner "Tri : Création" dans le dropdown
▶ Collapse ligne projet Nouveau
  • ▶
    Bouton ▶ dans la 1ère colonne de la ligne résumé du projet
  • 👆
    Clic → masque toutes les tâches du projet (gain de place)
  • ▶
    ▶ horizontal = tâches masquées / ▶ tourné 90° = tâches visibles
  • 💾
    État sauvegardé dans localStorage entre les sessions
📦 Archivage ZIP
  • 📦
    Bouton 📦 dans l'en-tête → dossier complet zippé dans archives/
  • 📂
    Le ZIP contient tout : JSON, images, STL, Cura, docs
  • 👁
    Section Archives sidebar pour voir les projets archivés
  • 🔄
    Restaurer — recrée le dossier depuis le ZIP
  • ✅
    Sélection multiple + "Supprimer la sélection"
⚪ À faire 🔵 En cours 🟢 Terminé 🔴 Bloqué 🟡 En attente

Gestion des tâches

Les tâches ont 2 niveaux de sous-tâches. Le temps se saisit directement en heures et minutes. Les tâches terminées descendent automatiquement en bas.

📋 Champs d'une tâche
  • 🔴
    Statut — clic direct sur la pill dans le tableau
  • 🎯
    Priorité — Haute, Moyenne, Basse
  • 📅
    Début / Deadline
  • ⏱
    Estimation — format libre : 2h 30min, 45min...
  • 🔔
    Rappel — notification Windows à date/heure précise
  • 📝
    Notes — icône 📝 dans le tableau si une note existe
  • 🏷
    Tags — dans la modale détail
  • 📎
    Pièces jointes — images base64 avec lightbox
  • 💬
    Commentaires — horodatés, éditables, Ctrl+Entrée
🕐 Saisie du temps H/Min
  • 🕐
    Colonne TEMPS : deux champs H et Min saisissables directement dans le tableau
  • ⏱
    Colonne PASSÉ : total calculé (tâche + toutes ses sous-tâches)
  • 📊
    Total visible dans la barre du bas et le Dashboard
  • 📁
    La ligne projet affiche la somme de toutes ses tâches
▶ Collapse sous-tâches Nouveau
  • ▶
    Bouton ▶ à gauche du nom de chaque tâche ayant des étapes
  • 👆
    Clic → masque/affiche les sous-tâches de cette tâche
  • 🔄
    ▶ horizontal = sous-tâches masquées / tourné = visibles
  • 💡
    Utile pour les tâches avec beaucoup d'étapes
📑 Sous-tâches (2 niveaux)
  • ➕
    Survol tâche → bouton + Étape
  • ✅
    Cocher la parente coche toutes les sous-tâches
  • 🔴
    Toutes sous-tâches terminées → parente terminée automatiquement
  • ⬇
    Les tâches terminées descendent automatiquement en bas

Gestionnaire de fichiers

Chaque projet stocke ses fichiers physiquement dans 4 sous-dossiers. Le drag & drop trie automatiquement par extension.

📁 Les 4 dossiers
  • 📷
    images/ — .jpg .png .gif .webp .svg
  • 🖨
    stl/ — .stl .obj .3mf .step .stp
  • ⚙
    cura/ — .gcode .curaproject .cura
  • 📄
    docs/ — .pdf .txt .docx .xlsx .ino .py .zip...
⬆ Upload auto-trié
  • 📁
    Bouton 📁 dans l'en-tête du projet
  • ⬆
    Glisser-déposer les fichiers sur la zone pointillée
  • ✅
    Extension détectée → bon dossier automatiquement
  • 📷
    Images : miniatures cliquables (lightbox)
  • ⬇
    Bouton ⬇ télécharger, 🗑 supprimer
💡

Accès direct : Tu peux copier des fichiers directement dans data/Nom du projet/images/ depuis l'Explorateur Windows — ils apparaîtront automatiquement.

Liste de composants

Chaque projet peut avoir une liste de composants/matériel avec quantité, fournisseur, prix unitaire et lien. Total calculé automatiquement.

🔧 Accéder aux composants
Cliquer sur le bouton 🔧 dans l'en-tête du projet. Un compteur s'affiche quand des composants sont définis.
📋 Colonnes disponibles
  • 📛
    Composant — ex: Arduino Mega 2560
  • 🔢
    Quantité
  • 🏪
    Fournisseur — AliExpress, Amazon...
  • 💶
    Prix unitaire (€)
  • 📊
    Total ligne — calculé auto (qté × prix)
  • 🔗
    Lien URL — vers le produit
  • 📝
    Remarque — notes libres

Sauvegarde NAS automatique Nouveau

GP sauvegarde automatiquement tes données sur ton NAS Synology après chaque modification. Aucune configuration nécessaire — tout est automatique.

💾 Indicateur topbar
  • 💾
    💾 NAS — état initial
  • 🔄
    🔄 NAS... — sauvegarde en cours (thread séparé)
  • ✅
    ✅ NAS — succès (survol = heure de la dernière sauvegarde)
  • ❌
    ❌ NAS — erreur (NAS injoignable, réseau coupé...)
  • 👆
    Clic sur le bouton = sauvegarde manuelle immédiate
⚙ Configuration
  • 📄
    Fichier backup_config.json dans le dossier du projet
  • 🔒
    Jamais envoyé sur GitHub (sécurité mot de passe)
  • 📂
    Copie data/ et archives/ vers le NAS
  • ⚡
    Non bloquant — tourne en arrière-plan
  • 🔄
    Vérification statut toutes les 60 secondes
⚠

NAS injoignable ? Si l'icône ❌ s'affiche, vérifie que le NAS est allumé et accessible sur le réseau. La sauvegarde se relancera automatiquement à la prochaine modification.

Diagnostic de cohérence

Le diagnostic analyse tous les projets actifs et signale les incohérences. Un clic sur une alerte ouvre directement la modale pour corriger.

❌ Erreurs critiques
  • ❌
    Projet "Terminé" mais des tâches encore ouvertes
  • ❌
    Tâche terminée mais des sous-tâches non terminées
⚠ Alertes
  • ⚠
    Projet sans aucune tâche
  • ⚠
    Tâche(s) en retard
  • ⚠
    Budget dépassé
  • ⚠
    Temps passé > estimation +20%
  • ⚠
    Projet bloqué sans description
ℹ Informations
  • ℹ
    Tâches sans deadline
  • ℹ
    Tâches sans estimation de temps
  • ℹ
    Composants sans prix
👆

Navigation directe : Cliquer sur une alerte ferme le diagnostic et ouvre la modale correspondante. Les projets/tâches "En attente" sont exclus des alertes.
