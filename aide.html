🚀 Demarrage rapide
Gestionnaire de projets local inspire de Monday.com, specialise pour les projets Arduino, Raspberry Pi et logiciels.
1
Lancer le serveur
Double-cliquer sur start.bat dans le dossier du projet. Une fenetre noire s'ouvre — ne pas la fermer tant que tu utilises l'appli.
2
Ouvrir dans le navigateur
Le navigateur s'ouvre automatiquement sur http://localhost:8742
3
Creer un projet
Cliquer sur + Nouveau projet ou appuyer sur la touche N
4
Ajouter des taches
Cliquer sur + Tache dans l'en-tete du projet ou appuyer sur T
Conseil : Les donnees sont sauvegardees automatiquement dans projets.json. Ce fichier est synchronise via OneDrive si tu as configure ce chemin.
👁 Les 5 vues
Chaque vue offre une perspective differente sur tes projets et taches.
📋 Tableau B
Vue principale. Affiche tous les projets avec leurs taches en lignes. Cliquer sur une tache pour ouvrir le detail. En-tete de projet collant pour faciliter la navigation.
📁 Kanban K
4 colonnes : A faire, En cours, Termine, Bloque. Glisser-deposer les projets entre les colonnes pour changer leur statut rapidement.
📅 Calendrier
Vue mensuelle avec les deadlines des projets et taches. Navigation mois par mois. Cliquer sur un evenement pour ouvrir le projet/tache.
📈 Timeline / Gantt
Barres horizontales representant la duree de chaque projet et tache. Ligne rouge = aujourd'hui. Necessite une date de debut et une deadline.
📊 Dashboard D
Tableau de bord avec statistiques globales : statuts, priorites, avancement par projet, taches en retard, budget, suivi du temps et journal d'activite.
📁 Gestion des projets
Chaque projet peut avoir des taches, des composants, un budget, un schema de cablage et des tags.
📝 Informations de base
  • 👁
    Nom — Editable directement dans le tableau (clic sur le nom)
  • 📦
    Type — Arduino/RPI, Logiciel, Mixte, Divers
  • 🔴
    Statut — A faire, En cours, Termine, Bloque (clic sur la pill)
  • 🎯
    Priorite — Haute, Moyenne, Basse
  • 📅
    Dates — Date de debut et deadline
  • 📄
    Description — Notes, liens, objectifs
⚙ Actions disponibles
  • 🔄
    Dupliquer — Copie le projet et toutes ses taches
  • 🗃
    Archiver — Masque le projet sans le supprimer. Visible via "Archives" dans la sidebar
  • 📄
    Exporter PDF — Export complet avec taches, sous-taches et notes
  • 🔥
    Supprimer — Suppression definitive avec confirmation
📈 Schema de cablage
Dans la modale projet, champ "Schema de cablage". Utiliser le bouton Parcourir pour selectionner une image — elle est stockee en base64 et persiste. Cliquer sur "Schema" dans l'en-tete pour agrandir en lightbox.
💰 Budget
Deux champs dans la modale : Budget estime et Budget reel. Le dashboard affiche un widget avec barre de progression et alerte si le budget est depasse. Le total des composants est additionne automatiquement.
🏷 Tags / Etiquettes
Ajouter des tags dans la modale projet. Tags predefinies : impression3D, electronique, mecanique, firmware, urgent, test, debug, etc. Ou creer des tags personnalises. S'affichent dans l'en-tete du projet.
✅ Gestion des taches
Les taches sont les etapes d'un projet. Elles peuvent avoir des sous-taches sur 2 niveaux.
📝 Champs d'une tache
  • 📝
    Titre — Nom de la tache
  • 🔴
    Statut — A faire, En cours, Termine, Bloque (clic sur la pill dans le tableau)
  • 🎯
    Priorite — Haute, Moyenne, Basse
  • 📅
    Debut et Deadline — Dates de la tache
  • ⌛
    Estimation — Duree prevue en heures
  • 🔔
    Rappel — Notification Windows a une date/heure precise
  • 🔗
    Tache parente — Pour creer une sous-tache d'une tache existante
  • 📄
    Notes — Details techniques, references, liens
  • 🏷
    Tags — Etiquettes dans la modale detail
🕐 Chronometre
  • ▶
    Cliquer sur ▶ dans la colonne Temps pour demarrer
  • ⏹
    Cliquer sur ⏹ pour arreter et sauvegarder
  • 💾
    Sauvegarde automatique toutes les 60 secondes
  • 📊
    Total visible dans le Dashboard (Suivi du temps)
📄 Sous-taches (2 niveaux)
  • ➕
    Survol une tache → bouton + Etape vert
  • 🔗
    Ou choisir une tache parente dans la modale
  • ✅
    Cocher la tache parente coche toutes les sous-taches
  • 🔴
    Toutes les sous-taches terminees = tache parente terminee automatiquement
  • ➕
    Ajouter une etape a une tache terminee la repasse en cours
💬 Commentaires
Dans la modale detail d'une tache (bouton "Ouvrir" au survol) : section Commentaires en bas avec date/heure automatique. Modifier ou supprimer chaque commentaire. Ctrl+Entree pour envoyer rapidement.
Rappels : Cliquer sur 🔔 dans la topbar pour autoriser les notifications. Ensuite definir une date/heure dans le champ "Rappel" d'une tache. La page doit rester ouverte.
🔧 Liste de composants
Chaque projet Arduino/RPI peut avoir une liste de composants avec quantite, fournisseur, prix et lien.
🔧 Acceder aux composants
Dans l'en-tete du projet, cliquer sur le bouton 🔧 pour ouvrir la vue composants. Un compteur s'affiche quand des composants sont definis.
📋 Champs par composant
  • 📝
    Nom — Ex: Arduino Mega 2560
  • 🔢
    Quantite — Nombre de pieces
  • 🏫
    Fournisseur — Ex: AliExpress, Amazon
  • 💰
    Prix unitaire — En EUR
  • 🔗
    Lien URL — Lien direct vers le produit
💰 Total automatique
Le total est calcule automatiquement (quantite x prix) et affiche en bas de la liste. Ce total est aussi visible dans le widget Budget du Dashboard.
⚓ Raccourcis clavier
Ces raccourcis fonctionnent quand le curseur n'est pas dans un champ de saisie.
Touche Action Details
N Nouveau projet Ouvre la modale de creation de projet
T Nouvelle tache Dans le projet courant (dernier survole ou clique)
D Vue Dashboard Tableau de bord global
B Vue Tableau Vue Board principale
K Vue Kanban Vue colonnes glissables
? Aide raccourcis Affiche la liste des raccourcis dans l'appli
Echap Fermer Ferme toute modale ou popup ouverte
Astuce : Dans la topbar, le bouton ? affiche aussi les raccourcis directement dans l'application.
💡 Astuces et conseils
📊 Filtres et tri
  • 🔍
    La barre de recherche cherche dans les noms de projets, taches ET sous-taches
  • 🔴
    La sidebar filtre par statut (A faire, En cours...) ou par type (Arduino, Logiciel...)
  • 📋
    Le tri (Deadline/Priorite/Nom) s'applique aussi aux taches dans chaque projet
  • ✅
    Bouton Terminees pour masquer les taches cochees
🌙 Mode sombre
Cliquer sur le toggle rond dans la topbar pour basculer entre mode clair et sombre. La preference est sauvegardee et restauree au prochain lancement.
🕐 Historique des versions
  • 🕐
    Bouton 🕐 dans la topbar pour ouvrir l'historique
  • 💾
    Sauvegarde automatique a chaque modification
  • 5️⃣
    Les 5 dernieres versions sont conservees
  • ↻
    Cliquer sur Restaurer pour revenir a une version anterieure
  • ⚠
    La restauration ecrase la version actuelle
📊 Statistiques avancees
  • 📊
    Visibles en bas du Dashboard
  • ⌛
    Temps passe par projet (barres)
  • 🔴
    Repartition des taches par statut
  • 📁
    Projets par type (Arduino, Logiciel...)
  • 🔴
    Taches en retard avec nombre de jours
  • 📈
    Avancement global par projet en SVG
📄 Export PDF
  • 📄
    Bouton PDF dans la topbar : exporte tous les projets visibles
  • 📄
    Bouton PDF dans l'en-tete d'un projet : exporte ce projet uniquement
  • 📐
    Inclut taches, sous-taches, statuts, deadlines, temps estime/passe et notes
💾 Sauvegarde
Toutes les donnees sont sauvegardees automatiquement dans projets.json. Ce fichier est sur OneDrive donc synchronise entre tes machines. Le chronometre se sauvegarde toutes les 60s automatiquement.
🚀 Workflow recommande
  • 1️⃣
    Creer le projet avec type, priorite et deadline
  • 2️⃣
    Ajouter les composants et le schema
  • 3️⃣
    Decomposer en taches principales puis en etapes
  • 4️⃣
    Utiliser le chronometre pour tracker le temps
  • 5️⃣
    Suivre l'avancement dans le Dashboard
🔗 Liens entre projets
  • ✏
    Dans la modale ✎ Modifier d'un projet, section "Liens avec d'autres projets"
  • 🔗
    Depend de (orange) — ce projet ne peut demarrer que si l'autre est avance
  • 📄
    Reference (bleu) — lien informel entre deux projets connexes
  • ↓
    Les projets dependants s'affichent automatiquement en dessous de leurs dependances dans le tableau
  • 👁
    Les liens sont visibles dans l'en-tete du projet, cliquables pour ouvrir le projet lie
⚠ Limitations connues
  • 🔔
    Les rappels necessitent que la page reste ouverte
  • 📈
    Le schema local doit etre charge via Parcourir pour persister
  • 💾
    Les images en base64 grossissent le fichier projets.json
  • 🔗
    Les fichiers locaux non-image ne peuvent pas etre affiches dans le navigateur
